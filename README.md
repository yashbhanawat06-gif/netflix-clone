# 🎬 Netflix Clone (HTML & CSS)

A simple Netflix landing page clone built using only HTML and CSS.  
This project focuses on replicating Netflix's UI with clean layout and styling.

---

## 🚀 Features
- Netflix-inspired user interface  
- Hero banner section  
- Movie rows with thumbnails  
- Clean and modern layout  
- Basic responsiveness  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  

---

This project is a Netflix UI clone built for educational purposes.
